# @transmute/did-key-resolver

- [Swagger Docs](https://did-key.web.app/api/docs)

## Getting Started

```
npm run serve:local
```

### Resolve a DID

```
curl -s -X GET \
http://localhost:5010/api/dids/did:key:z6MkspKCfo45Ak8YfodDW14oasR4bXSGAHEjAwJDvNdicvQt \
| jq '.'
```
